# `@cva/tailwindcss`

For documentation, visit [cva.style](https://cva.style).

## Installation

```sh
npm i cva @cva/tailwindcss tailwind-merge
```

## Usage

Same behaviour as `cva`, but consuming via:

```ts
import { cva, cx } from "@cva/tailwindcss";
```
